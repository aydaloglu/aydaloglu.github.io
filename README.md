# Modern Portfolio

A minimalist, responsive portfolio website built with React, Vite, and Tailwind CSS.
Modeled after [sedagenbasi.github.io](https://sedagenbasi.github.io/).

## Getting Started

### Prerequisites

- Node.js (version 16 or higher)

### Installation

1. Open your terminal in this directory.
2. Install dependencies:
   ```bash
   npm install
   ```

### Running the Project

Start the development server:
```bash
npm run dev
```
Open [http://localhost:5173](http://localhost:5173) to view it in the browser.

### Customization

- **Personal Info**: Update `src/components/Hero.jsx` and `src/components/Contact.jsx`.
- **Experience**: Edit the `experiences` array in `src/components/Experience.jsx`.
- **Projects**: Edit the `projects` array in `src/components/Projects.jsx`.
- **Styling**: Tailwind configuration is in `tailwind.config.js` and global styles in `src/index.css`.

## Technologies

- [React](https://reactjs.org/)
- [Vite](https://vitejs.dev/)
- [Tailwind CSS](https://tailwindcss.com/)
- [Lucide React](https://lucide.dev/) (Icons)
